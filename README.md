# sudoku-shell
A sudoku game that achieved through the shell script.
# usage
* Terminal

    ./sudoku.sh <br>
    or <br>
    ./sudoku.sh 1 <br>
    or <br>
    ./sudoku.sh 8 <br>
* explain

    option: from 1 to 8, means the game level.
# 简介   
一种用shell脚本实现的终端数独游戏。
# 使用说明
* 终端运行sudoku.sh脚本

    ./sudoku.sh <br>
    或 <br>
    ./sudoku.sh 1 <br>
    或 <br>
    ./sudoku.sh 8 <br>
    
* 说明

    脚本参数指游戏等级，共有8个等级（1~8）。
* 操作

    上、下、左、右键移动光标，数字键输入，'f'完成游戏，'q'退出游戏。
